#### LectPro — Final Project Management System

A role-based academic project management platform built as a final project at Kinneret College (2024).  
Designed for students, instructors, and department heads to track and manage final academic projects — with stage management, file uploads, a two-way comment system, role-based access control, and department isolation.

`React 19` `Node.js` `Express` `MySQL` `JWT` `Multer` `Nodemailer`
